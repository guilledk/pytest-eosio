API Reference
=============

typing
******

.. _typing_exe_result:

ExecutionResult
---------------
Typing alias returned from functions which execute a command using the container
shell.

Under the hood: ``Tuple[int, str]``

    1) Exitcode
    2) ``STDOUT`` & ``STDERR`` muxed as a string.

.. _typing_exe_stream:

ExecutionStream
---------------
Typing alias returned from functions which launch a command using the container
shell. This functions don't wait for the command to finish and instead return a
consumable stream.

Under the hood: ``Tuple[str, Iterator[str]]``

    1) Docker internal process id. 
    2) ``STDOUT`` & ``STDERR`` muxed as a consumable iterator.

.. _typing_tx_result:

TransactionResult
-----------------
Typing alias for a dictionary generated by the json returned of a executed
blockchain transaction.

Under the hood: ``Dict[str, Union[str, int]]``

.. _typing_action_result:

ActionResult
------------
Typing alias returned from functions which push an action to the blockchain.

Under the hood: ``Tuple[int, Union[TransactionResult, str]]``

    1) Exitcode of ``cleos push action`` shell execution.
    2) ``TransactionResult`` when transaction is correctly executed or if exitcode is not ``0``, a string with the command's ``STDOUT`` & ``STDERR``.
